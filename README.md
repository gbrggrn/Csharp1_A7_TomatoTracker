## Functionality

* Add tomatoes
  - Breed
  - Plant date
  - Growth type (tall, low, ample, bush)
  - Other notes
* View a dashboard of:
  - To do today (water, trellis)
  - Watering dates for next 3 weeks
  - Trellising dates for next 3 weeks
  - Projected harvest dates
* Save dashboard (TXT Serialization)
* Load dashboard
* New dashboard
* Help/exit

---

## Setup & Usage

1. Clone the repository
2. Open in IDE of your choice
3. Register your tomato breeds
4. Plan your watering and trellising
5. Enjoy homegrown tomatoes!
